Introduction to ML

Machine Learning enables systems to learn from data and make predictions.

Examples include image recognition, language translation, and self-driving cars.

It relies on data and algorithms to improve over time without explicit programming.

import sklearn
Supervised vs Unsupervised Learning

Supervised Learning uses labeled data for training.

Unsupervised Learning works with unlabeled data, finding patterns.

  • Supervised Learning: Examples include classification and regression.
  • Unsupervised Learning: Examples include clustering and association.
  • model.fit(X_train, y_train)

Choosing the right method depends on the data and the problem type.

Model Evaluation

Evaluating a model is essential to understand its performance.

  • Accuracy: Percentage of correct predictions.
  • Precision and Recall: Useful for imbalanced datasets.
  • F1 Score: Balances precision and recall.

Cross-validation can ensure reliability across datasets.

from sklearn.model_selection import cross_val_score
Common Algorithms

Popular machine learning algorithms include:

  • Linear Regression
  • Logistic Regression
  • Decision Trees
  • Support Vector Machines
  • K-Nearest Neighbors

Each algorithm has its own strengths for different types of data.

model = LinearRegression()
Overfitting and Underfitting

Overfitting: Model performs well on training data but poorly on new data.

Underfitting: Model is too simple and fails to capture patterns in the data.

Finding the right balance is crucial for a good model.

model = RandomForestClassifier()
Data Preprocessing

Data Preprocessing is the first step in building a machine learning model.

Common steps include handling missing values, scaling, and splitting the dataset.

Feature scaling ensures features contribute equally to the model.

Data is often split into training and testing sets for evaluation.

X_train, X_test, y_train, y_test = train_test_split(X, y)
